# lingxia-sdk

LingXia cross-platform SDK for native integrations

## Supported Platforms

- **Android** - Android SDK (Kotlin/Java)
- **Apple** - iOS/macOS SDK (Swift/Objective-C)
- **HarmonyOS** - OpenHarmony SDK (ArkTS)

```
lingxia-sdk/
├── android/     # Android SDK implementation
├── apple/       # iOS/macOS SDK implementation
└── harmony/     # HarmonyOS SDK implementation
```
